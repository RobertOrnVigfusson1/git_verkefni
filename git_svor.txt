Git Spurningar og svör

1. Hvað gera eftirfarandi Linux skipanir?
	cd = færir þig milli mappa, breytir skrám
	cd ~ = skilar þér beint á heimasvæðið
	ls = sýnir lista af skrám/file-um/möppum
	pwd = prentar út hvar þú ert
	mkdir = býr til skrá



2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone = býr til copy af miðuðu geymslu
	git log = sýnir hvenær þú komst inná eða gerðir einhvað
	git status = sýnir stöðu vinnuskrár og svæðisbundin svæði
	git diff = skoðar öll samruna
	git checkout = skiptir um allskonar mið


3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff = til að sjá hvað þú ert búinn að gera en sýnir ekki hvað þú ert búinn að stage-a
	b) git diff --staged = sýnir hvað þú ert búinn að stage-a
	c) git diff commit1 commit2 = sýnir hversu öðruvísi þau eru

	4. 	Hvað er útgáfustýring (e.version control)? = útgáfustýring gerir það að tölvan þín update-ar alltaf í nýjasta stýrikerfið.


	5.	Hverjir eru helstu kostir við að nota GIT? =  getur unnið án netsambands, getur unnið með öðrum að gera verkefni, version control í git er mjög gott.

	6.	Hversu oft telur þú að eigi að gera commit í verkefni, rökstuddu? =átt að gera commit við hverja einustu spurningu eða verkefni.
